## 6. Removal of Unreliable Regex-Based Extraction Tools

**Summary:** The MCP server had 3 extraction tools (extract_facts, extract_timeline, extract_biography) that relied on regex pattern matching for structured data extraction. These tools suffered from low accuracy (~60-70%), false positives, and inability to understand context. Removed these unreliable tools and their helper functions, updated dependent quiz tools to use only extract_article, and created comprehensive documentation of the removed implementations for future reference. Goal was to maintain codebase quality by removing unreliable functionality while preserving implementation details for potential future recreation with proper NLP/ML methods.
